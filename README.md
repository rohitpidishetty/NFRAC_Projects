## NFRAC Projects

Proprietary API possessed by National Federation for Research in Advanced Computing (NFRAC), helps in populating content on NFRAC application.
## https://nfrac-in.web.app/nfrac_projects

This repository also contains the career end-points for all the divisions at NFRAC. 

For each wing, there is a careers.json file inside Careers/ folder.

New openings shall be updated here, just make sure to update the JSON and the jobs at the users end will be automatically sorted (Recent first).

This is how a job should be posted.

```
{
    "id": "",
    "heading": "",
    "title": "",
    "location": "",
    "type": "",
    "status": "",
    "postedAt": "",
    "shortDesc": "",
    "longDesc": "",
    "responsibilities": []
  }
```
